🌐 Architecture Overview

Data Forge Plus runs on a Flask + AWS backend. Files are uploaded by users to an S3 bucket, cleaned by a Lambda function using an OpenAI-powered agent pipeline, and results are stored in a cleaned S3 prefix. RDS tracks user/file metadata. Users then click "Explore" to launch the Streamlit-powered Data Forge Lite for visual exploration.

🚀 Status

✅ MVP complete – fully deployed with EC2 + Gunicorn, RDS integration, and Lambda workflows tested. Currently adding CI/CD and multi-agent routing improvements.
