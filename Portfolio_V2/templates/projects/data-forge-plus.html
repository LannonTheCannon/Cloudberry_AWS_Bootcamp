Data Forge Plus is your personal AI-powered data scientist. It transforms messy, unstructured CSVs into clean, usable insight—no manual wrangling required. Upload up to five datasets, and watch as the system detects patterns, engineers features, and summarizes your data using real intelligence.

🚀 Launch the Demo 📦 View Source on GitHub

🔍 What You Can Do

  • AI-Powered Cleaning: Leverages GPT-4o-mini to analyze and fix messy, inconsistent datasets.
  • Feature Engineering: Automatically creates new columns that enhance model performance.
  • Upload and Track: Supports multi-file uploads (up to 5) with per-user logging via AWS S3 and RDS.
  • Explore Visually: Use Data Forge Lite (Streamlit) to mind-map and preview datasets in real-time.

🧠 Tech Stack

  • Flask + Gunicorn (API layer)
  • EC2 with Gunicorn and CI/CD pipeline
  • S3 (raw & cleaned storage)
  • RDS (MySQL user & file metadata)
  • Streamlit (exploration interface)
  • OpenAI API (GPT-4o-mini) & LangChain
  • Tailwind CSS (styling + layout)

🌐 How It Works

Once a user logs in, they can upload one or more CSV files to their personal S3 bucket. These files are processed by an OpenAI-powered agent pipeline running inside AWS Lambda, which performs both data cleaning and feature engineering. Cleaned results are stored separately, and metadata is logged in RDS.

Users can then enter “exploration mode” — launching Streamlit Lite — where they view a smart sample of the dataset, ask follow-up questions, and receive AI-curated insights or code recommendations.
