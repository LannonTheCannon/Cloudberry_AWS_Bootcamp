Exo Explorer README

By Lannon Khau

Jun 13, 2025 · 6 min read
Exo Explorer is a concept-stage AI platform that visualizes exoplanets and predicts their habitability based on real NASA datasets.

🧠 About the Project

Exo Explorer bridges data science with astronomy. It ingests the NASA Exoplanet Archive, performs predictive modeling on planetary attributes, and uses generative AI to render what these alien worlds might look like. It's storytelling through science.

🔧 Architecture Overview

  • ✅ FastAPI backend for NASA ingestion + ML pipeline
  • ✅ Streamlit dashboard for front-end visual exploration
  • ✅ XGBoost & Scikit-learn for habitability modeling
  • ✅ LangChain agents for prompt logic
  • ✅ DALL·E 3 & Stability AI for planet rendering

📁 Project Structure

  • 📄 exo_pipeline.py — ML + inference logic
  • 📄 render_planet.py — Image prompt generator
  • 📄 explorer_dashboard.py — Streamlit app interface
  • 📂 /static/images — Teasers and results

⚙️ Running Locally

git clone https://github.com/LannonTheCannon/Exo-Explorer.git
cd Exo-Explorer
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
streamlit run explorer_dashboard.py
