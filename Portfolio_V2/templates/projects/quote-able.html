Quote-Ability README

By Lannon Khau

Jun 12, 2025 · 4 min read
Quote-Ability is a personal reading companion—an AI-powered Streamlit app designed to help readers recall, reflect on, and summarize what they've read.

🧠 About the Project

The inspiration? Forgetting the best parts of what you've read. Quote-Ability fixes that by letting you upload a PDF and talk to your book. Get contextual summaries, revisit prior sections, or ask questions—all without spoilers.

🔧 Architecture Overview

  • ✅ Streamlit frontend with file uploader + chat input
  • ✅ PyPDFLoader + LangChain for text processing
  • ✅ GPT-4o mini for natural language interaction
  • ✅ ChromaDB with pysqlite3 for vectorstore persistence
  • ✅ Streamlit Secrets for secure API key injection

📁 Project Structure

  • 📄 ebook_tutor.py — Main Streamlit app
  • 📄 requirements.txt — pip dependencies
  • 📄 .streamlit/secrets.toml — API key storage
  • 📄 Practice_Folder_V1/data/ — Chroma persistence

⚙️ Running Locally

git clone https://github.com/LannonTheCannon/Octopus_Books.git
cd Octopus_Books
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
streamlit run ebook_tutor.py

🚧 Limitations

- PDF uploads are capped at ~200MB
- ChromaDB may encounter sqlite issues if not patched with pysqlite3
- Does not yet support scanned PDFs or OCR (coming soon!)

🛣️ Roadmap

  • ✅ File upload + RAG chat enabled
  • ✅ Streamlit Cloud deployment support
  • ⏳ OCR support for image-based PDFs
  • ⏳ Bookmarking and save-to-library options
  • ⏳ Multi-user authentication and storage via S3
